## Image Optimization

Before adding images to this directory:

1. **Compress images:**
   - Use tools like TinyPNG, ImageOptim, or Squoosh
   - Target: < 200KB per image

2. **Use appropriate formats:**
   - PNG for images with transparency
   - JPG for photos
   - SVG for icons and illustrations
   - WebP for modern browsers (with fallback)

3. **Provide retina versions:**
   - Consider @2x versions for high-DPI displays
   - Or use SVG for sharp scaling

## Creating the Bird Illustration

If you need to create the SouvenirBird illustration:

### Design Guidelines:
- **Style:** Minimalist, geometric, friendly
- **Colors:** Primarily mint (#7EC4A8) with lighter tints
- **Character:** Small bird, round shapes, warm expression
- **Details:** Simple features, no complexity
- **Format:** SVG for scalability
